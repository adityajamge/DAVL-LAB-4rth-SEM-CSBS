Theory

1. Titanic Dataset Overview

The Titanic dataset contains information about passengers aboard the RMS Titanic, including survival status, passenger class, age, gender, fare, and other attributes. It is commonly used for classification and data analysis tasks.

2. Types of Data

Data Type

Description

Examples from Titanic

Numerical (Continuous)

Values that can take any number

age, fare

Numerical (Discrete)

Countable whole numbers

sibsp, parch, pclass

Categorical (Nominal)

Categories without order

sex, embarked

Categorical (Ordinal)

Categories with order

pclass (1st, 2nd, 3rd)

3. Visualization Types

· Histogram: Shows distribution of a single numerical variable

· Box Plot: Shows median, quartiles, and outliers

· Violin Plot: Combines box plot with kernel density estimation

· Pair Plot: Shows relationships between all pairs of variables

· Heatmap: Shows correlation matrix with color coding

4. Outlier Detection

Outliers are detected using the IQR (Interquartile Range) method:

· IQR = Q3 - Q1

· Lower Bound = Q1 - 1.5 x IQR

· Upper Bound = Q3 + 1.5 x IQR

· Values outside these bounds are considered outliers

import seaborn as sns

import matplotlib.pyplot as plt

import pandas as pd

import numpy as np

 

print("Name: Aditya Babanrao Jamge")

print("PRN Number: RBTL25CB076")

 

# Load the Titanic dataset

titanic = sns.load_dataset('titanic')

print(f"Shape: {titanic.shape}")

print(titanic.head())

 

# 1. IDENTIFY FEATURES AND CLASSIFY BY DATA TYPE

numerical_features = titanic.select_dtypes(include=[np.number]).columns.tolist()

categorical_features = titanic.select_dtypes(include=['object', 'category', 'bool']).columns.tolist()

 

print("\nNumerical Features:", numerical_features)

print("Categorical Features:", categorical_features)

 

# 2. HISTOGRAMS AND BOX PLOTS

fig, axes = plt.subplots(2, 3, figsize=(15, 10))

fig.suptitle('Histograms of Numerical Features')

for idx, col in enumerate(numerical_features[:6]):

    ax = axes[idx // 3, idx % 3]

    titanic[col].hist(ax=ax, bins=20, edgecolor='black')

    ax.set_title(col)

plt.tight_layout()

plt.show()

 

# Box plots

fig, axes = plt.subplots(2, 3, figsize=(15, 10))

for idx, col in enumerate(numerical_features[:6]):

    ax = axes[idx // 3, idx % 3]

    titanic.boxplot(column=col, ax=ax)

    ax.set_title(col)

plt.tight_layout()

plt.show()

 

# Outlier detection

for col in numerical_features:

    Q1 = titanic[col].quantile(0.25)

    Q3 = titanic[col].quantile(0.75)

    IQR = Q3 - Q1

    outliers = titanic[(titanic[col] < Q1 - 1.5*IQR) | (titanic[col] > Q3 + 1.5*IQR)]

    print(f"{col}: {len(outliers)} outliers")

 

# 3. SEABORN VISUALIZATIONS

# Box plots by category

sns.boxplot(data=titanic, y='age', x='survived')

plt.title('Age by Survival')

plt.show()

 

# Violin plots

sns.violinplot(data=titanic, y='age', x='survived')

plt.title('Age Distribution by Survival')

plt.show()

 

# Pair plot

sns.pairplot(titanic[['survived', 'pclass', 'age', 'fare']].dropna(), hue='survived')

plt.show()

 

# Correlation heatmap

plt.figure(figsize=(10, 8))

correlation_matrix = titanic[numerical_features].corr()

sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm', center=0)

plt.title('Correlation Heatmap')

plt.show()

 

# Missing values visualization

plt.figure(figsize=(12, 6))

sns.heatmap(titanic.isnull(), cbar=True, yticklabels=False, cmap='viridis')

plt.title('Missing Values Heatmap')

plt.show()

 

# 4. DISTRIBUTION COMPARISON

sns.histplot(data=titanic, x='age', hue='survived', kde=True)

plt.title('Age Distribution by Survival')

plt.show()

 

# 5. CHART TYPE RECOMMENDATION TABLE

recommendations = pd.DataFrame({

    'Data Type': ['Numerical', 'Numerical', 'Categorical', 'Cat + Numerical'],

    'Purpose': ['Distribution', 'Correlation', 'Frequency', 'Comparison'],

    'Charts': ['Histogram, Box Plot', 'Heatmap, Scatter', 'Bar, Pie', 'Violin, Box']

})

print(recommendations)

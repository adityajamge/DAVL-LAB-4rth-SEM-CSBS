Title

School Student Marks Analysis System using Python

Aim

To develop a Python program that:

  • Stores student names and marks using variables, lists, and tuples
  • Calculates total, average, and percentage for each student
  • Determines pass/fail status and assigns grades (A: greater than or equal to 80, B: 60-79, C: 40-59, F: less than 40)
  • Identifies students with highest and lowest marks and prints all even marks
  • Performs result analysis including mean, median, mode, and standard deviation for every course

Theory

1. Python Data Structures

Variables: Used to store individual values like subject names.

Lists: Ordered, mutable collections used to store student names and marks.

Tuples: Ordered, immutable collections used to store fixed data like subjects and individual student records.

2. Grading System

Grade Percentage Range Status
A 80% and above Pass (Excellent)
B 60% - 79% Pass (Good)
C 40% - 59% Pass (Average)
F Below 40% Fail

3. Statistical Measures for Result Analysis

  • Mean: Average of all marks in a subject
  • Median: Middle value when marks are sorted
  • Mode: Most frequently occurring mark
  • Standard Deviation: Measure of spread around the mean

# SCHOOL STUDENT MARKS ANALYSIS SYSTEM

# 1. STORE STUDENT NAMES AND MARKS USING VARIABLES, LISTS, AND TUPLES

# Using variables for subjects
subject1 = "Mathematics"
subject2 = "Science"
subject3 = "English"
subject4 = "History"
subject5 = "Computer"

# Using tuple for subjects (immutable)
subjects = (subject1, subject2, subject3, subject4, subject5)

# Using list for student names
students = []

# Using list of tuples for storing marks
student_marks = []

# Input student data
print("=" * 60)
print("        STUDENT MARKS ENTRY SYSTEM")
print("=" * 60)

num_students = int(input("Enter the number of students: "))

for i in range(num_students):
    print(f"\n--- Student {i + 1} ---")
    name = input("Enter student name: ").strip()
    students.append(name)
    
    marks = []
    for subject in subjects:
        while True:
            try:
                mark = float(input(f"  Enter marks in {subject} (0-100): "))
                if 0 <= mark <= 100:
                    marks.append(mark)
                    break
                print("  Please enter marks between 0 and 100.")
            except ValueError:
                print("  Invalid input.")
    
    student_marks.append((name, tuple(marks)))

# 2. CALCULATE TOTAL, AVERAGE, AND PERCENTAGE

student_results = []

for name, marks in student_marks:
    total = sum(marks)
    average = total / len(marks)
    percentage = (total / (len(marks) * 100)) * 100
    
    student_results.append({
        'name': name, 'marks': marks, 'total': total,
        'average': average, 'percentage': percentage
    })

# 3. DETERMINE PASS/FAIL STATUS AND ASSIGN GRADES

def get_grade(percentage):
    if percentage >= 80: return "A"
    elif percentage >= 60: return "B"
    elif percentage >= 40: return "C"
    else: return "F"

def get_status(percentage):
    return "PASS" if percentage >= 40 else "FAIL"

for result in student_results:
    result['grade'] = get_grade(result['percentage'])
    result['status'] = get_status(result['percentage'])

# 4. IDENTIFY HIGHEST/LOWEST MARKS AND PRINT EVEN MARKS

highest_student = max(student_results, key=lambda x: x['total'])
lowest_student = min(student_results, key=lambda x: x['total'])

all_marks = []
for result in student_results:
    all_marks.extend(result['marks'])

even_marks = [int(mark) for mark in all_marks if int(mark) % 2 == 0]

# 5. RESULT ANALYSIS - MEAN, MEDIAN, MODE, STANDARD DEVIATION

def mean(ar): return sum(ar) / len(ar)

def median(ar):
    sorted_ar = sorted(ar)
    n = len(sorted_ar)
    mid = n // 2
    if n % 2 == 0:
        return (sorted_ar[mid - 1] + sorted_ar[mid]) / 2
    return sorted_ar[mid]

def mode(ar):
    frequency = {}
    for i in ar:
        frequency[i] = frequency.get(i, 0) + 1
    max_freq = max(frequency.values())
    modes = [key for key, value in frequency.items() if value == max_freq]
    if len(modes) == len(frequency):
        return "No mode"
    return modes

def std_dev(ar):
    m = mean(ar)
    variance = sum((x - m) ** 2 for x in ar) / len(ar)
    return math.sqrt(variance)

# Calculate statistics for each subject
for i, subject in enumerate(subjects):
    subject_marks_list = [result['marks'][i] for result in student_results]
    print(f"\n{subject}:")
    print(f"  Mean: {mean(subject_marks_list):.2f}")
    print(f"  Median: {median(subject_marks_list):.2f}")
    print(f"  Mode: {mode(subject_marks_list)}")
    print(f"  Standard Deviation: {std_dev(subject_marks_list):.2f}")
